# Architecture Diagram

Browser ↓ Frontend (React SPA)

**Components:**
- Vite (Bundler)
- React (UI Library)
- Wouter (Routing)
- Tailwind CSS (Styling)

**Data Source:**
- Local static JSON/JS files in `src/data/` folder.

This application is entirely client-side. No backend APIs, external databases, or authentication layers exist in this repository.
